# Data

`TB_real_data.RData` (~38 MB) is the primary input to the pipeline. It is not tracked in git but can be generated by running `scripts/2_TB_getdata.R`. The script requires the following files, which **are** tracked in this repo:

| File | Cohort | Source |
|---|---|---|
| `combined_sub.RData` | Africa — GSE79362 (Zak et al. 2016), edgeR/TMM normalized | Originally from a public RNA-seq processing pipeline |
| `GSE107994/GSE107994_edgeR_normalized_Leicester_with_progressor_longitudinal.xlsx` | India proxy — GSE107994 (Leong et al. 2018) | GEO supplementary |
| `GSE107994_sample_info.csv` | India proxy sample metadata | Derived from GEO series matrix |

The remaining cohorts (USA, South Africa, Malawi) are downloaded from NCBI GEO automatically by the script.

## Generating TB_real_data.RData

```bash
# 1. Install environment
SKLEARN_ALLOW_DEPRECATED_SKLEARN_PACKAGE_INSTALL=True pixi install
pixi run install-all

# 2. Run from the repo root
pixi run Rscript scripts/2_TB_getdata.R
```

Output: `data/TB_real_data.RData` — 6 cohorts × 10,695 genes, binary TB/LTBI labels.

## Note on combined_sub.RData

The Africa cohort was processed through an edgeR/TMM RNA-seq pipeline externally. The original processing code is not available in this repository. `combined_sub.RData` is included as a fixed input so the full pipeline is reproducible from this repo without re-running that processing step.
